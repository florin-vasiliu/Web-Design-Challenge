Project Description

The objective of the project was to answer a fundamental question: "What is the weather like as we approach the equator?"

The data used for this analysis has been collected from http://api.openweathermap.org using an application programming interface (commonly referred to as API).

In order to identify weather trends as a function of location on the globe, the parameters selected for characterizing weather were temperature, humidity, cloudiness, and wind speed, while the places used for weather data collection were cities selected on the whole range latitudes, and random longitudes.
